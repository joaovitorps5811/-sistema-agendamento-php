# Sistema de Agendamento para Barbearia

## Descrição

Este projeto tem como objetivo o desenvolvimento de um sistema de agendamento para uma barbearia, permitindo o controle de horários, clientes e serviços. O sistema está sendo desenvolvido utilizando Programação Orientada a Objetos com PHP e integração com banco de dados MySQL.

## Objetivo

Criar um sistema simples onde clientes possam realizar agendamentos, administradores possam gerenciar usuários e horários e o sistema organize os serviços oferecidos pela barbearia.

## Modelagem

A modelagem do sistema foi baseada nos conceitos de classes, objetos, herança e relacionamentos entre entidades. O diagrama desenvolvido combina características de UML com elementos de modelagem de banco de dados, utilizando tipos como varchar e enum para aproximar a estrutura da implementação real.

## Estrutura do Projeto

Agendamento/

index.php - Página inicial
cadastro.php - Cadastro de usuários
login.php - Login do sistema
agendamento.php - Tela de agendamento
README.md - Documentação do projeto

## Banco de Dados

O banco utilizado se chama sistema_agendamento e contém tabelas como usuarios, agendamentos e servicos.
